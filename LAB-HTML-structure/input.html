3.ให้หาว่า input type มีอะไรบ้าง และแต่ละ type แตกต่างกันอย่างไร

input attribute differentiation
button text or alert , href สร้างปุ่มสำหรับกดเพื่อลิงค์ไปยังภายนอก
checkbox text or id name value สำหรับสร้างตัวเลือกแบบ ติ๊กถูกโดยสามารถจำกัดจำนวนตัวเลือกได้หรือไม่จำเป็นต้องเลือกก็ได้เช่นกัน
color text or id name สำหรับเลือกสีต่างๆ โดยหลังจากเลือกสีนั่นๆจะโชว์บนตัว Element
date text or id name min-max ตัวเลือกวัน เดือน ปี สามารถกำหนดขอบเขต จำกัดช่วงเวลาได้
datetime text or id name value min-max ตัวเลือกวัน เดือน ปี และเวลา สามารถกำหนดขอบเขต จำกัดช่วงเวลาได้
email text or id name กล่องข่้อความสำหรับกรอก email สร้างข้อจำกัดเพื่อกรอก email เท่านั่น
file text or id name ปุ่มเพื่อสร้าง browser สำหรับการอัพโหลดไฟล์
hidden text or id name value สร้างตัวเลือกสำหรับเลือกเดือนและปี สามารถกำหนดขอบเขตระยะเวลาได้
image text or src,id,name,width,height สร้างกล่องสำหรับใส่รูปภาพ โดยสามารถจำกัดขนาดของรูปภาพได้
month text or id,name สำหรับสร้างตัวเลือก เดือนและปี สามารถจำกัดขอบเขตระยะเวลาได้
number text or id,name,min-max สร้างตัวเลือกสำหรับใส่จำนวนเป้นตัวเลข โดยสามารถกำหนดขอบเขตระหว่างตัวเลขนั่นๆได้
password text or id,name สำหรับสร้างกล่องใส่ Password โดยจะซ่อน text ที่เราพิมพ์ (สามารถแสดงได้เช่นกัน)
radio text or id name value สำหรับสร้างตัวเลือกแบบติ๊ก โดยมีข้อจำกัดเลือกได้อย่างใดอย่างหนึ่ง
range text or id,name,value min-max สร้างปุ่มป้อนค่าตัวโดยตัวเลือกควบคุมแบบสไลด์ สามารถกำหนดค่าต่ำสุดและสูงสุดได้
reset text or id name value สำหรับรีเซ็ทค่าที่อยู่แบบฟอร์มทั่งหมด
search text or id,name for searching สร้างกล่องสำหรับพิมพ์ข้อความเพื่อใช้ค้นหาไปยังลิงค์ภายนอก
submit text or id name value or link กำหนดปุ่มสำหรับยื่นแบบฟอร์มไปสู่ตัวจัดการแบบฟอร์ม
tel text or id name pattern กล่องข่้อความสำหรับใส่เบอร์โทรศัพท์สามารถกำหนดรูปแบบตัวเลขได้
text text or id.name กล่องใส่ข้อความ สามารถจำกัดจำนวนตัวอักษรได้
time text or id name value สร้างตัวเลือกเวลา ชม นาที วินาที สามารถกำหนดขอบเขตของช่วงเวลาได้
url text or id,name กล่องข้อความสำหรับใส่ ลิงค์ URL Address
week text or id name value ตัวเลือกสำหรับเลือกจำนวนสัปดาห์ เดือน และปี สามารถกำหนดขอบเขตตัวเลือกได้
