Selector Exemple

1.simple selector = ประกอบไปด้วย Element, id, class, group และ Universal

1.1 Element Selector คือ element ที่เป็นองค์ประกอบพื้นฐานที่ใช้ใน HTML เช่น P, DIV, SPAN เป็นต้น

1.2 id คือ การกำหนด Attribute เพื่อเป็นองค์ประกอบเฉพาะของ HTML นั่นๆ โดยตัวอักษรที่ใช้ประกอบขึ้นมาต้องไม่ซ้ำกับองค์ประกอบหลัก โดยจะใส่ # นำหน้าตัวอักษรในการเรียกใช้งาน เช่น ID=mystyle ในส่วนการใช้งานจะเป็น #mystyle{Property:valuel;}

1.3 Class ใช้งานเหมือน ID เพียงแต่ใช้ . แทน #

1.4 group คือ การใช้งานหลายๆ element selector ในคราวเดียวเช่น h1, h2, p { Property:value; }

1.5 Universal คือ การใช้ element ที่จะส่งผลกับทุกองค์ประกอบในหน้า HTML โดยตรง โดยใช้ * เป็นตัวแทนของ element ในการกำหนด เช่น * {Margin: 0;} (คือการเริ่มต้นหน้า HTML ด้วย margin=0%)

Cobinator Exemple

1.แบบ Descendant Selector คือการกำหนด element selector แบบมากกว่า 2 selector โดยมีเงื่อนไขว่า selector ดังกล่าวต้องเป็นลูกหลานกัน โดยใช้การเว้นวรรค เช่น Div h1 {property:value;} เป็นต้น

2.แบบ Child Selector คือการกำหนด element selector ที่อยู่ภายใต้เงื่อนไขว่าเป็นลูกของ element หลัก โดยใช้เครื่องหมาย > เป็นตัวบ่งชี้ เช่น Div > h1 คือ การเลือก element h1 ที่ต้องเป็นลูกหลานของ element div เท่านั่น

3.แบบ Adjacent Sibling Selector (+) คือการเลือก element selector ที่อยู่ในระดับพี่น้องในตำแหน่งถัดไป โดยจะเป็นการเลือก element ที่อยู่ถัดไปของ element ก่อนหน้า โดยใช้ + เป็นเครื่องหมายบ่งชี้ เช่น div+p จะหมายถึง P selector ที่อยู่ถัดไปจาก element div เท่านั่น

4.แบบ General Sibling Selector (~) คือการเลือก element selector ที่อยู่ในระดับพี่น้อง ถัดไปทั้งหมด โดยใช้ตัว (~) ในการบ่งชี้ เช่น div ~ p หมายถึง p element ที่อยู่ถัดจาก div ทั้งหมด คือ selector

Property Value
คุณลักษณะ ค่าที่กำหนด

กลุ่มของ Property

1.Property for content คือคุณลักษณะ ที่เปลี่ยนแปลง content นั่นๆ เช่น สี,ขนาด,แบบตัวอักษร เป็นต้น

2.Property for border คือคุณลักษณะตัวกล่อง,โดยรอบหรือด้านใดด้านหนึ่งของรูปแบบกล่อง Element

3.Property for padding คือคุณลักษณะขนาดระหว่าง content และ border

4.Property for margin คือคุณลักษณะขนาดระหว่าง boder และ ขอบเขตของ html

5.Property for element คือคุณลักษณะของการจัดวางรูปแบบของ element

Unit ของ Value

1.Absolute lenghts คือ ค่าที่กำหนดแบบตายตัวตามหน่วยที่ใช้งาน เช่น cm,px,mm เป็นต้น

2.relative lenghts คือ ค่าที่กำหนดโดยอ้างอิงจากองค์ประกอบอื่นๆ เช่น em(อ้างอิงตามขนาดฟอนต์),ex(ตามความสูงของฟอนต์) เป็นค้น
