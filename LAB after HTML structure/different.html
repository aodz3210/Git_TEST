2.ให้หาตวามเหมือนและแตกต่างระหว่าง tags ที่กำหนด

Tag 1 Tag 2 Tag 3 ความเหมือนและความแตกต่าง
hr br - hr สำหรับสร้างเส้นแนวนอนใต้ Element และ Br สำหรับหยุด Element และขึ้นบรรทัดใหม่
ul ol - ul ใช้สร้างลิสรายการโดยลำดับจะถูกกำหนดด้วยสัญลักษณ์ • ส่วน ol จะสร้างลำดับรายการโดยมีตัวเลขกำกับ
select checkbox radio button select ใช้สำหรับสร้างตัวเลือกแบบลิสรายการที่เลื่อนลงมาด้านล่าง checkbox สำหรับสร้างตัวเลือกแบบ ติ๊กถูกโดยสามารถจำกัดจำนวนตัวเลือกได้หรือไม่จำเป็นต้องเลือกก็ได้เช่นกัน Radio สำหรับสร้างตัวเลือกแบบติ๊ก โดยมีข้อจำกัดเลือกได้อย่างใดอย่างหนึ่ง
div span - div สำหรับทำให้กล่องข้อความอยู่บรรทัดถัดไป Span เว้นวรรคกล่องข้อความแต่ยังอยู่ในบรรทัดเดียวกันกับกล่องข้อความก่อนหน้า
script style - script ใช้เพื่อฝั่งสคริปฝั่งไคลเอ็นต์ ส่วน style ใช้สำหรับฝังรูปแบบ เช่นสีต่างๆลงในเอกสาร
