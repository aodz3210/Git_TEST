Tag attribute การใช้งาน
< h> h1-h6 ใช้เป็นหัวข้อเรื่อง โดยมีอักษรขนาดใหญ่และค่อยๆเล็กลงตามลำดับ 1-6
< p> text paragraphs ใช้สำหรับใส่ข้อความย่อหน้าทั่วๆไป
< div> text ใช้เพื่อให้กล่องข้อความลงไปอยู่บรรทัดถัดไป
< img> src , alt สำหรับใส่รูปภาพโดย Src สำหรับใส่แหล่งที่มา และ alt สำหรับใส่ข้อความบ่งบอกลักษณะภาพ
< a > href เพื่อสร้างลิงค์ไปยังภายนอก ใช้ href เพื่อระบุ address
< ul> li เพื่อสร้าง list รายการโดยด้านหน้าจะเป็นรูปกลมๆกำกับ (สามารถใช้ CSS)
< ol> li เพื่อสร้าง list รายการโดยด้านหน้ากำกับด้วยตัวเลข
table tr.th.td ใช้เพื่อนสร้างตาราง tr สำหรับเพิ่มแถว th ใช้สำหรับหัวข้อหรือทำให้ตัวอักษรเป็นตัวหนา td ใช้สำหรับใส่ตัวอักษรขนาดปกติ
form input ใช้สำหรับการสร้างแบบฟอร์มโดยรูปแบบขึ้นอยู่กับคำสั่ง input
buttom text,a(link) ใช้สำหรับสร้างปุ่มกดโดยสามารถกำหนดข้อความบนปุ่มและแนบลิงค์ภายนอกได้
