Display Property values Description
1.inline แสดงผล element เป็นแบบ inline (ทำงานเหมือน span) ทำให้การปรับขนาดไม่มีผล
2.block แสดงผล element เป็นแบบ block (ทำงานเหมือน P) สามารถปรับขนาดได้
3.content ลบ container ที่คลุม element ออก และทำให้ child element ถัดไปรวมกัน
4.flex แสดงผล element เป็นแบบ block level และ มี 1 block layout ไม่ว่าจะเป็นแนวตั้งหรือแนวนอน
5.grid แสดงผล element เป็นแบบ block level และ สามรถมีหลายๆ block layout ใน 1 container
6.inline-block แสดงผล element แบบ block ในรูปแบบ inline โดยที่เราสามารถปรับขนาดได้
7.inline-flex ใช้งานเหมือน flex แต่จะเปลี่ยน element เป็นรูปแบบ inline แทน
8.inline-grid ใช้งานเหมือน grid แต่จะเปลี่ยน element เป็นรูปแบบ inline แทน
9.list-item ทำให้ element เรียงกันเหมือน tag li
10.run-in ให้การแสดงผลอ้างอิงจากการใช้งาน สามารถเปลี่ยนรูปแบบได้ทั้ง inline และ block
11.table ทำให้ element เรียงกันเหมือน tag table
12.table-caption แสดงผล element เหมือน tag caption
13.table-column-group แสดงผล element เหมือน tag colgroup
14.table-header-group แสดงผล element เหมือน tag thead
15.table-footer-group แสดงผล element เหมือน tag tfoot
16.table-row-group แสดงผล element เหมือน tag tbody
17.table-cell แสดงผล element เหมือน tag td
18.table-column แสดงผล element เหมือน tag col
19.table-row แสดงผล element เหมือน tag tr
20.none ลบ element
21.initial ทำให้ element นั่นๆแสดงผลแบบค่า default
22.inherit ทำให้ element นั่นๆแสดงผลแบบตาม parent


Grid Flex
สามารถกำหนด layout แบบ 2มิติได้ด้วย column และ row มีอสิระในการจัดวาง layout ใน container วาง layout แบบ 1มิติ คือวาง layout แบบ block เดียว ไม่ว่าจะแนวตั้งหรือแนวนอน


Justify Align
การจัดวาง layout ใน Container นั่นๆในแนวนอน (แกน X) การจัดวาง layout ใน Container นั่นๆในแนวตั้ง (แกน Y)
